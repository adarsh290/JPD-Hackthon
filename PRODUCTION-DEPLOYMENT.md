## Deployment Process

### 1. Pre-Deployment Verification ✅

- **Build Process**: Clean `tsc` compilation to dist/

### 2. Render Deployment Steps

#### Option A: Blueprint Deployment (Recommended)
```bash
# 1. Push to GitHub
git add .
git commit -m "Production-ready hardened configuration"
git push origin main

# 2. Deploy via Render Blueprint
# - Go to render.com dashboard
# - New → Blueprint
# - Connect GitHub repository
# - Apply configuration
```

#### Option B: Manual Service Creation
1. **Database**: Create PostgreSQL service first
2. **Backend**: Create Web Service with hardened config
3. **Frontend**: Create Static Site with optimized build

### 3. Build Sequence (Automated by Render)

#### Backend Build Process
```bash
# 1. Install dependencies (including devDependencies for build)
npm install

# 2. Auto-generate Prisma client (via postinstall)
prisma generate

# 3. Compile TypeScript to JavaScript
npm run build  # Executes: tsc

# 4. Start production server
npm start  # Executes: node dist/server.js
```

#### Frontend Build Process
```bash
# 1. Install dependencies
npm install

# 2. Build React application
npm run build  # Vite production build

# 3. Serve static files from ./dist
```
